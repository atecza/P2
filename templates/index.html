Countries and their Ecological Footprint

Project Overview

About the Data

This project was undertaken as part of the University of Denver's Data Science Bootcamp. Visualizations were generated using two datasets: Ecological data was obtained from the Global Footprint Network in conjuntion with Mathis Wachernagel and William Reese at the University of British Columbia, economic data was provided by the World Bank.

An "Ecological Footprint" measures the ecological assets a given population requires to produce the natural resources it consumes and to absorb its waste (e.g. CO2). A nations biocapacity represents the productivity of its ecological assets. If left unharvested, these areas container can help absorb waste such as CO2. Both the ecological footprint and biocapacity are expressed in global hectares. A global hectare is a hectare standardized with world average productivity. If a populations ecological footprint exceeds a region's biocapacity, that region runs an ecological deficit.

Coding Info

This project focused on three aspects of working in the d3 library: multiple data sources, updating, and responsiveness. The map interface was created by nesting ecological data inside the geojson callback function. This allowed subsequent graphs to be generated when a path (country) within the map was clicked. Updating in this project was achieved in two ways. The first, demonstrated by the world data graph, utilized transitions. The second, seen in the country bar graph, relied on clearing the data from an svg and then repopulating. To achieve full responsiveness, the viewbox function in d3 was combined with bootstrap.
